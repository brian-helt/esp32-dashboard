About My Project

My name is Brian Helt.

I started this project to learn more about Microcontrollers, C++, and Web Development. As the project goes one, I'll be working on custom PCBA design as well.

My patio gets very little sunlight, which makes it difficult for me to learn how to grow plants and start a garden. So I thought 'what if I get a sesnor that measures the amount of light my plants are getting? There are a few types of plant monitoring devices that you can but that all have fancy apps, but I figured why can't I make one myself?

Prototype Stage

The 'prototpye' stage of the LeafPanel project makes use of ESP32 devices. They are similar to Arduinos, but more powerful and have WiFi built in. The idea is that I can have one ESP32 (or more, for gathering data from multiple plants) that has various sensors connected to it. The initial sensors I chose are an ambient light sensor, a moisture sensor, and a temperature/humidity sensor, currently all sourced from DFRobot.com. These 'secondary' ESP32's can gather all the data from those sensors and send it to the 'primary' ESP32 using something called ESP-NOW, which is a communication system specific to ESP32 devices. The 'primary' ESP32 is connected to the internet over WiFi, and sends the data to my web server hosted on Render.com. Hosting via Render gives me a secure public web server, that lets me see live data from my device from anywhere.
